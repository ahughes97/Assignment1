Copyright, Creative Common (CC) Licenses, Fair Use

Copyright is a form of intellectual property which gives the creator of original work the right to copy, publish, transmit, and adapt their own material. (Copyright Licensing New Zealand, 2022).

Basic principles of Copyright are:
1. Copyright applies automatically when something is published.
2. The Copyright symbol acts as a reminder therefore when it isn't displayed, Copyright still applies.
3. Everything published is protected by Copyright including print material, computer programmes, music, and page layouts.
4. Copyright lasts more than a lifetime. In NZ, literary, dramatic, artistic, and musical works stay in place for the lifetime of the creator plus 50 years after their death. (Copyright Licensing New Zealand, 2022).
Reference

Creative Commons (CC) Licenses are constructed by combining four license elements:
1. ShareAlike - Any adaptation of work must be licensed under the same license.
2. Attribution - Guarantees that the creators are given proper credit.
3. NoDerivatives - If a user creates an adaptation of the work, it may not be shared.
4. NonCommercial - Prohibits the use of content for commercial purposes. (Regents of the University of Colorado, 2022).

Privacy Policy

A privacy policy is a legal document which disclosed to the user the types of information collected, the reason why, and the methods used to collect personal information.
Personal information includes all the data that'll identify an individual person such as name, email address, and passport number. (Laird, 2022).
Reference

Search Engine Optimisation

There are multiple ways to improve the visibility of the website on the search engine. (McCormick, n/d).

1. Target keywords within the website - make sure that each page within the website clearly targets a specfic topic to be noticed more.
Conduct keyword research to find the high-volume, relevant keywords and incorporate them into the pages and meta tags.
2. Add the website to online directories - the more places the website is listed, the more frequently the website will get noticed.
Examples of online directories include Facebook, Instagram, LinkedIn, Google My Business, and Yelp.
3. Use compelling images - optimise high-quality photos in order to rank in an image search by choosing an appropriate file name, keeping the file size small to reduce load time, and add captions.
4. Use local media - reach out to local papers, magazines, radio stations, and news channels to direct traffic to the website.
Placing ads in newspapers and local publications will help bring in newcomers to the website.
5. Write quality content - to ensure the quality of the content on the website, be sure to dive deep into topics, include images, videos, and other media, aim to provide an experience for the users, and monitor the load times. (McCormick, n/d).

Web Publishing

There are many factors to consider when evaluating a web hosting provider

1. Average uptime - this is the average amount of time the website stays up within a given time period. The uptime is important as it can affect how much traffic the website gets. The website won't be getting much traffic if users can't access it. The average uptime should generally be 100% or as clase to it as possible.
2. Load time - this is the amount of time it takes for a web page to load. The load time affects how much traffic the website gets. The slower the web pages' load time, the more frustrated the users get. The web host should provide page load times of 3 seconds or less.
3. Support - getting helpful customer support means dealing with website problems faster and more effectively. Web hosting providers usually offer phone or chat customer support. It's important to have quick responses and knowledgeable team support. (McCormick, n/d).

Web Performance and Maintenance

Ways to reduce page loading time:
1. Compress and optimise the images - large images can delay loading times therefore reducing the image file size can reduce their "weight" and increase loading times.
2. Reduce the redirects - too many redirects on the website can impact the loading times. Eliminating unnecessary redirects on the website can result in lower page loading times.
3. Cache the web pages - Caching stores copies of the sites' files therefore minimising the work needed for the server to generate and serve a web page to a users' browser.
4. Enable browser caching - this enables the browser to store a variety of information including stylesheets, images, and JavaScript files therefore it doesn't have to reload the entire page every time a user visits it.
5. Use asynchronous and defer loading for CSS and JavaScript files - asynchronous loading enables multiple files to load at the same time which can speed up the pages' performance. (Duò, 2020).
Reference

Actions or tasks to carry out when maintaining a website:
1. Perform yearly usability tests to ensure the website continues to make sense for users across all browsers and devices.
2. At least once a year check the renewal status of the domain name for the website.
3. Update the header, footer, and legal policies to keep the information relevant, and to make sure the Terms of Service and Privacy Policy are up to date as laws change.
4. Test out all the forms on the website to ensure they are working on all browsers and devices, and that valuable information isn't missing from forms that aren't working correctly.
5. Fix any broken links every few months by either removing them or replacing them with updated links. (Rowland, 2021). Reference

Web Security

Common cyber attacks / threats and preventive measures against those attacks:

1. Password based attacks - hackers use techniques and tools to gain access to usernames and passwords including password guessing and password-cracking software.
Ways to prevent this attack include long passwords, unique passwords, and secure passwords.
2. Denial of service attacks - scammers send quantities of data to a website, rendering it unusable.
Ways to prevent this attack include constantly monitoring the website and network traffic to detect unusual occurrences, and keeping the systems up to date.
3. Social engineering attacks - a common type of this attack is phishing which involves emails alleging to be from trustworthy entities but have deceptive messages that aims to gain sensitive information.
Ways to prevent this attack include creating a security-conscious work culture, implementing two-factor authentication, using phishing detection tools, and using common sense.
4. Man in the middle attacks - occurs when a hacker inserts themselves into a communication session between two parties and eavesdrops to steal sensitive information.
Ways to prevent this attack include paying attention to security certificate warnings, and using secured networks.
5. Malware attacks - malicious software is designed to destroy or disrupt a computer. Common types of malware include viruses, worms, ransomware, and spyware.
Ways to prevent this attack include browsing cautiously, using good software, and acting quickly. (Adams, 2022).
